1.5 Describing copyright and other constraints affecting websites


If you post content that you have created there is no adverse copyright clause affecting your website. If you post content created by others, you could find yourself in legal wrangling if you failed to obtain permission from the copyright holder. If the permission to use is perpetual you can use it time and again. Also check to see if the permission to use is transferable.

1.7 Describe which are the appropriate filetypes for websites


Bmp, jpg, png and gif. Bmp is not ideal for websites. Jif is an 8 bit, 256 colour format which may be enough for what you require. Jpg supports compression and so it is ideal for use on the web, unlike bmp which does not support compression. Png is also a good choice for use on the web.
